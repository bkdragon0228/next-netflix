## 주요기능

* 비동기 통신
* 검색
* 페이지 라우팅

## 사용 기술

* React
* Next.js
* axios
* React query
* emotion

## Getting Started

설치
```bash
npm install
# or
yarn
```

실행
```bash
npm run dev
# or
yarn dev
```
